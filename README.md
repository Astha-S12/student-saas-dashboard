This project demonstrates full-stack web development skills including authentication, CRUD operations, cloud database integration, dashboard analytics, data visualization, CSV export, and deployment.

# 🚀 Installation

Clone the repository:

```bash
git clone YOUR_GITHUB_LINK
```

Move into the project directory:

```bash
cd student-saas-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
