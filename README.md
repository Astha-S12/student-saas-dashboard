# рҹҡҖ Installation

Clone the repository:

```bash
git clone YOUR_GITHUB_LINK
```

Move into the project directory:

```bash
cd student-saas-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

